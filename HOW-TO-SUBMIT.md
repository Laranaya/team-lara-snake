# How to Submit

---

## Option A – GitHub Repo _(recommended)_

1. Make sure all your changes are saved.
2. Push your branch to GitHub:

```
 git add .
 git commit -m "feat: submission"
 git push
```

3. **Add the following collaborators** to your repository:
   - Go to your repo on GitHub → **Settings** → **Collaborators** (under "Access")
   - Click **"Add people"** → search by email → click **"Add"**
   - Repeat for each person below:
     - Pavita.Koliar@nielseniq.com
     - GiimHorng.Tung@nielseniq.com
     - Oomkar.Pandit@nielseniq.com
     - Ankit.Nigam@nielseniq.com
     - Anil-Kumar.Balachandra@nielseniq.com
4. Fill in the **Submission Form** (link shared by your facilitator) with:
   - Your team name
   - Your GitHub repo URL (e.g. `https://github.com/your-username/team-[name]-snake`)
   - Confirm you have added all 5 collaborators

---

## Option B – OneDrive Link _(if GitHub is not possible)_

1. Make sure all your changes are saved.
2. Zip your entire project folder.
   - The folder should be named: `team-[your-team-name]-snake`
   - The ZIP should be named: `team-[your-team-name]-snake.zip`
3. Upload the ZIP file to your **OneDrive**.
4. Right-click the file → **Share** → Set access to **"People in NIQ"** → **Copy link**.
5. Fill in the **Submission Form** with:
   - Your team name
   - Paste your OneDrive sharing link
   - Confirm the link access is set to "People in NIQ"

## Pre-submission Checklist

Run through this before you submit:

- [ ] `snake.html` opens in a browser and the game is fully playable (snake moves, food appears, score updates)
- [ ] `evidence/phase1-chat.md` — filled in with your Chat exchange
- [ ] `evidence/phase2-plan.md` — filled in with your Plan mode output
- [ ] `evidence/phase3-prompt.md` — filled in with your best prompt + slash command usage
- [ ] `evidence/phase4-features.md` — lists at least 2 extra features with the prompts used
- [ ] Phase Progress Evidence: Git commits optional; ZIP teams may include phase-progress.md
- [ ] **GitHub teams:** All 5 collaborators added to the repository
- [ ] **OneDrive teams:** Sharing link set to "People in NIQ"
